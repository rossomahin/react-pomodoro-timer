# Pomodoro Timer

A simple Pomodoro Timer application with different modes: **Run**, **Short Break**, and **Long Break**.  
Track your cycles and manage your focus sessions efficiently.

---

## Tools

### React + TypeScript

- Zustand for state management
- TailwindCSS for styling
- Lucide Icons for buttons

---

## Features

- Timer with **Run**, **Short Break**, and **Long Break** modes.
- Automatic cycle tracking.
- Visual indicators for cycles.
- Responsive design, works on mobile and desktop.
- Persistent cycles using localStorage (powered by Zustand).

---

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/rossomahin/react-pomodoro-timer.git
cd react-pomodoro-timer
npm install
```
